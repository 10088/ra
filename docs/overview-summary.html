The ra application

=======

Ra

Version: 0.9.0

Authors: Team RabbitMQ.

ra is an implementation of the Raft consensus algorithm (https://raft.github.io/). It allows users to write persistent, replicated and fault-tolerant state machines by implementing the ra_machine behaviour.

The ra module provides the API for starting, stopping and deleting Ra clusters, managing membership as well as interacting with ra_machine implementations running inside ra clusters.
